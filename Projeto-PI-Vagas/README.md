# Vagas de Tecnologia

Página estática com exemplos de vagas na área de tecnologia.

Como usar:

1. Abra o arquivo `index.html` em um navegador (duplo-clique). 
2. Edite os cartões de vaga em `index.html` para ajustar títulos, locais e descrições.

Observações:
- A cor azul do modelo foi substituída por `#b20000` como solicitado.
- A fonte é sans-serif padrão do sistema. Para usar uma fonte específica (ex: Inter), adicione um link ao Google Fonts no `head` e atualize `font-family` em `styles.css`.

Imagem de fundo:
- Para adicionar uma imagem de fundo, coloque seu arquivo em `images/bg.jpg` dentro do projeto. O CSS já referencia `images/bg.jpg` por padrão.
- Se quiser usar outro caminho, edite a variável `--bg-image` em `styles.css` (no :root) e coloque o caminho entre aspas, ex: `--bg-image: url("assets/meu-bg.png");`.
